mars_time.constants#

The constants module is a collection of Martian orbital and temporal constants. Except where noted, these values come directly or indirectly from the Mars fact sheet.

Module Attributes

mars_year_0_start

Time of the start of Mars year 0.

martian_sol_length

Length of a Martian sol [hours].

orbital_eccentricity

Mars orbital eccentricity.

perihelion_sol

Sol number of perihelion.

seconds_per_sol

Number of seconds per Martian sol.

sols_per_martian_year

Number of sols per Martian year.
